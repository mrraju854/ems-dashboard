# 🚀 TaskFlow EMS – Employee Management System

A modern and responsive Employee Management System built with React and Tailwind CSS. TaskFlow EMS enables admins to assign tasks and employees to track their work status through an intuitive dashboard interface.

## ✨ Features

### 👨‍💼 Admin Dashboard

* Secure Admin Login
* Create and Assign Tasks
* View All Employee Tasks
* Monitor Task Distribution
* Real-time Dashboard Updates

### 👨‍💻 Employee Dashboard

* Secure Employee Login
* View Assigned Tasks
* Accept New Tasks
* Mark Tasks as Completed or Failed
* Track Task Statistics

### 📊 Task Management

* New Tasks
* Accepted Tasks
* Completed Tasks
* Failed Tasks
* Dynamic Task Counters

### 🎨 UI Features

* Fully Responsive Design
* Modern Dashboard Interface
* Clean Card-Based Layout
* Smooth User Experience
* Built with Tailwind CSS

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* Context API
* Local Storage
* JavaScript (ES6+)
* Vite
